EA untuk MT5 Valetax

HTML/browser tidak bisa klik order langsung ke Valetax MT5. Arsitektur aman: UI + Cloudflare Worker membuat sinyal, lalu EA MQL5 di terminal MT5 yang mengeksekusi order.
  1. Login ke akun Valetax MT5 langsung di terminal MT5: File → Login to Trade Account.
  2. Isi menu Login Akun MT5 di aplikasi ini untuk membuat input guard akun EA.
  3. Copy file mql5/ValetaxCloudflareAIBot.mq5 ke MQL5/Experts, lalu Compile di MetaEditor.
  4. MT5: Tools → Options → Expert Advisors, centang Allow WebRequest dan tambahkan URL Worker Cloudflare Anda.
  5. Attach EA ke chart symbol yang ingin dipakai. Set WorkerUrl, AppToken, risk, dan AllowAutoTrade.
  6. Uji pada akun demo minimal beberapa minggu sebelum live.
